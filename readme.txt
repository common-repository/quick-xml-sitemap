== Description ==

Easy and quickly generate an XML sitemap of your WordPress blog and ping the following search engines: Ask.com, Google, Bing. One click install and let go. XML will be update it-self daily. Special thanks to `Greg Molnar` for inspiring me with some


== Installation ==

1. Upload `quick-xml-sitemap` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
That`s all you need to do.
